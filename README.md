# Description

Backend service for Growthtop web application


# Used technologies

- [Nest](https://github.com/nestjs/nest) framework
- Typescript

# Installation

## Packages setup
Recomended package management with Yarn:

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start
```
